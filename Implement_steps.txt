# Deployment Database by Mongo Atlas
  1. Access mongodb.com/cloud/atlas -> Login by Google account
  2. Choose Create a deployment -> Choose M0 (Free)
  3. Chọn nhà cung cấp (AWS/Google Cloud/Azure) và khu vực (Region) gần bạn nhất (thường là Singapore để tối ưu tốc độ).
  4. Click Create
  5. Tạo username và password
  6. IP Access List -> Choose "Allow Access from Anywhere" or typing "0.0.0.0/0"
  7. Tại tab Database, nhấn nút Connect
  8. Chọn Drivers (Node.js / Python / ...)
  9. Copy đoạn mã có dạng: mongodb+srv://<username>:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority (Username và password lấy ở bước 5)
  10. Change password:
	- Vào mục Database Access ở menu bên trái
	- Edit -> Edit Password -> Update User
	- Cập nhật lại URL ở bước 9
  11. Ghi dữ liệu lên mongodb atlas bằng python script (Xem file seed.py)
